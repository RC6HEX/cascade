# Features (характеристики)

- Название приложения — «CurrencyX»
- Тёмная тема, акцентный цвет — зелёный (#10b981)
- Все надписи на русском языке
- Источник курсов — открытый API без ключа: `https://open.er-api.com/v6/latest/USD`
- Кеш курсов в localStorage с timestamp; срок жизни кеша — 1 час
- При offline / API недоступен — показывать пометку «Данные от {date}»
- Поддерживаемые валюты: USD, EUR, RUB, GBP, JPY, CNY, TRY, KZT, BYN, GEL, CHF, CAD
- Сумма обновляется по событию input (без кнопки «Конвертировать»)
- Никакого серверного кода — приложение полностью клиентское
