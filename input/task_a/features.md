# Features (характеристики)

- Название приложения — «QuickCalc»
- Тёмная тема по умолчанию, акцентный цвет — оранжевый
- Все надписи на русском языке
- При вводе невалидной операции поле подсвечивается красным
- Поддержка ввода с клавиатуры (цифры, +, -, *, /, Enter, Esc, Backspace)
- История вычислений отображается справа от калькулятора
- Никакого серверного кода — приложение полностью клиентское
