# Features (характеристики)

- Название приложения — «TaskBoard»
- Светлая тема, акцентный цвет — синий (#2563eb)
- Все надписи на русском языке
- Канбан-доска с тремя колонками: «К выполнению», «В работе», «Выполнено»
- Drag-and-drop для перемещения задач между колонками
- Данные хранятся в localStorage
- Кнопка фильтра показывает счётчик задач в каждом статусе
