# Нефункциональные требования

### НФТ-01: Отклик на действия пользователя
**Категория:** Производительность
**Описание:** Время отклика на действия пользователя (ввод чисел, выбор операций, нажатие кнопок) должно быть менее 100 мс.
**Источник:** БТ-01, БТ-03, БТ-04

### НФТ-02: Поддержка браузеров
**Категория:** Совместимость
**Описание:** Приложение должно корректно работать в последних версиях браузеров Chrome, Firefox, Safari и Edge.
**Источник:** БТ-05

### НФТ-03: Доступность интерфейса
**Категория:** Доступность (a11y)
**Описание:** Приложение должно соответствовать стандартам доступности WCAG 2.1 на уровне AA.
**Источник:** Features

### НФТ-04: Локализация
**Категория:** Локализация
**Описание:** Все надписи и сообщения в приложении должны быть на русском языке.
**Источник:** Features

### НФТ-05: Безопасность ввода
**Категория:** Безопасность
**Описание:** При вводе невалидных данных (например, букв вместо чисел) поле ввода должно подсвечиваться красным цветом, и пользователь не должен иметь возможности выполнить вычисление.
**Источник:** Features

### НФТ-06: Работа в автономном режиме
**Категория:** Надёжность
**Описание:** Приложение должно продолжать работать корректно после первой загрузки, даже при отсутствии подключения к интернету.
**Источник:** Features

### НФТ-07: История вычислений
**Категория:** Удобство использования
**Описание:** История последних 10 вычислений должна отображаться справа от калькулятора и обновляться в реальном времени.
**Источник:** БТ-02

### НФТ-08: Поддержка клавиатуры
**Категория:** Удобство использования
**Описание:** Пользователь должен иметь возможность вводить числа и выполнять операции с помощью клавиатуры (цифры, +, -, *, /, Enter, Esc, Backspace).
**Источник:** Features
