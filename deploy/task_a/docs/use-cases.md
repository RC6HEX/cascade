# Юз-кейсы

### UC-01: Выполнение базовых арифметических операций
**Источник:** БТ-01
**Актор:** Пользователь
**Предусловие:** Пользователь открыл веб-страницу калькулятора.
**Основной поток:**
1. Пользователь вводит первое число, нажимая на цифровые кнопки.
2. Пользователь выбирает операцию (+, −, ×, ÷).
3. Пользователь вводит второе число.
4. Пользователь нажимает «=».
5. Система отображает результат на экране.
**Альтернативный поток — деление на ноль:**
- 5a. Если второе число равно нулю и выбрана операция деления, система отображает сообщение «Ошибка: деление на ноль».
**Постусловие:** Результат вычисления отображается на экране, или сообщение об ошибке, если произошло деление на ноль.

### UC-02: Просмотр истории вычислений
**Источник:** БТ-02
**Актор:** Пользователь
**Предусловие:** Пользователь открыл веб-страницу калькулятора и выполнил несколько вычислений.
**Основной поток:**
1. Пользователь видит историю последних 10 вычислений, отображаемую справа от калькулятора.
**Постусловие:** История вычислений отображается на экране.

### UC-03: Очистка текущего ввода и сброс калькулятора
**Источник:** БТ-03
**Актор:** Пользователь
**Предусловие:** Пользователь открыл веб-страницу калькулятора и ввёл данные.
**Основной поток:**
1. Пользователь нажимает кнопку «C».
2. Система очищает текущий ввод.
**Альтернативный поток — полный сброс:**
- \*. В любой момент пользователь нажимает кнопку «CE».
- \*. Система полностью сбрасывает калькулятор.
**Постусловие:** Текущий ввод очищен или калькулятор полностью сброшен.

### UC-04: Обработка деления на ноль
**Источник:** БТ-04
**Актор:** Пользователь
**Предусловие:** Пользователь открыл веб-страницу калькулятора и ввёл данные для деления.
**Основной поток:**
1. Пользователь вводит первое число.
2. Пользователь выбирает операцию деления.
3. Пользователь вводит второе число, равное нулю.
4. Пользователь нажимает «=».
5. Система отображает сообщение «Ошибка: деление на ноль».
**Постусловие:** Сообщение об ошибке отображается на экране.

### UC-05: Доступность через веб-браузер
**Источник:** БТ-05
**Актор:** Пользователь
**Предусловие:** Пользователь имеет доступ к веб-браузеру.
**Основной поток:**
1. Пользователь открывает веб-страницу калькулятора в браузере.
2. Система загружает интерфейс калькулятора.
**Постусловие:** Интерфейс калькулятора доступен через веб-браузер.
