# Юз-кейсы

### UC-01: Выполнение базовой арифметической операции
**Источник:** БТ-01, БТ-04
**Актор:** Пользователь
**Предусловие:** Калькулятор открыт и готов к вводу.
**Основной поток:**
1. Пользователь вводит первое число, нажимая на цифровые кнопки.
2. Пользователь выбирает операцию (+, −, ×, ÷).
3. Пользователь вводит второе число.
4. Пользователь нажимает «=».
5. Система отображает результат на экране.
**Альтернативный поток — Деление на ноль:**
5a. Если второе число равно нулю и выбрана операция деления, система отображает сообщение «Ошибка: деление на ноль».
**Постусловие:** Результат вычисления (или сообщение об ошибке) отображен на экране.

### UC-02: Очистка ввода и сброс калькулятора
**Источник:** БТ-03
**Актор:** Пользователь
**Предусловие:** Калькулятор открыт, и в поле ввода есть данные (число, операция, результат) или калькулятор находится в состоянии ошибки.
**Основной поток:**
1. Пользователь нажимает кнопку «C».
2. Система очищает поле ввода.
**Постусловие:** Поле ввода очищено, калькулятор сброшен в исходное состояние.

### UC-03: Открытие веб-страницы калькулятора
**Источник:** БТ-05
**Актор:** Пользователь
**Предусловие:** Пользователь имеет доступ к веб-браузеру.
**Основной поток:**
1. Пользователь открывает веб-страницу калькулятора.
2. Система отображает интерфейс калькулятора.
**Постусловие:** Веб-страница калькулятора отображена, приложение готово к работе.
