# Функциональные требования

### ФТ-01: Ввод чисел и выполнение операций
**Источник:** БТ-01, UC-01
**Описание:** Система позволяет пользователю вводить числа и выполнять базовые арифметические операции
**Входные данные:** 
- Первое число (целое/дробное, положительное/отрицательное)
- Операция (+, -, *, /)
- Второе число (целое/дробное, положительное/отрицательное)
**Ожидаемый результат:** 
- Корректный результат операции отображается на экране
**Граничные случаи:**
- Ввод нуля в качестве одного из чисел
- Ввод отрицательных чисел
- Ввод очень больших чисел (1e308)
- Ввод дробных чисел с точкой и запятой (3.14 и 3,14)
- Ввод чисел в экспоненциальной записи (1e5)
- Последовательное выполнение операций без нажатия "=" (3+4*2)

### ФТ-02: Обработка деления на ноль
**Источник:** БТ-04, UC-04
**Описание:** Система корректно обрабатывает попытку деления на ноль
**Входные данные:** 
- Любое число в качестве делимого
- Ноль в качестве делителя
**Ожидаемый результат:** 
- Отображается сообщение "Ошибка: деление на ноль"
- Поле ввода подсвечивается красным
**Граничные случаи:**
- Деление нуля на ноль (0/0)
- Деление очень большого числа на ноль (1e308/0)
- Деление отрицательного числа на ноль (-5/0)

### ФТ-03: Очистка ввода и сброс калькулятора
**Источник:** БТ-03, UC-03
**Описание:** Система предоставляет возможность очистки текущего ввода и полного сброса
**Входные данные:** 
- Нажатие кнопки "C" (очистка) или "CE" (сброс)
**Ожидаемый результат:** 
- При нажатии "C": очищается текущий ввод
- При нажатии "CE": полностью сбрасывается состояние калькулятора
**Граничные случаи:**
- Нажатие "C" при пустом вводе
- Нажатие "CE" в середине вычисления
- Многократное нажатие кнопок сброса

### ФТ-04: Отображение истории вычислений
**Источник:** БТ-02, UC-02
**Описание:** Система отображает историю последних 10 вычислений
**Входные данные:** 
- Выполнение любых операций калькулятора
**Ожидаемый результат:** 
- В правой части экрана отображается список последних 10 операций с результатами
- Новые операции добавляются в начало списка
- При превышении 10 записей самые старые удаляются
**Граничные случаи:**
- История пуста (при первом открытии)
- Ровно 10 операций в истории
- Очистка истории после сброса калькулятора

### ФТ-05: Ввод с клавиатуры
**Источник:** Features
**Описание:** Система поддерживает ввод данных с клавиатуры
**Входные данные:** 
- Нажатия цифровых клавиш (0-9)
- Нажатия операций (+, -, *, /)
- Нажатия специальных клавиш (Enter, Esc, Backspace)
**Ожидаемый результат:** 
- Цифры добавляются в текущий ввод
- Операции выбираются как при клике мышью
- Enter выполняет вычисление (=)
- Esc очищает ввод (C)
- Backspace удаляет последний символ
**Граничные случаи:**
- Нажатие недопустимых символов (буквы, спецсимволы)
- Комбинации клавиш (Ctrl+V и т.д.)
- Долгое нажатие клавиш

### ФТ-06: Валидация ввода
**Источник:** Features
**Описание:** Система проверяет корректность вводимых данных
**Входные данные:** 
- Любые пользовательские данные (мышь/клавиатура)
**Ожидаемый результат:** 
- Некорректные данные (буквы, символы) игнорируются
- Поле ввода подсвечивается красным при ошибке
- Вычисление невозможно при невалидных данных
**Граничные случаи:**
- Ввод нескольких точек/запятых в числе
- Ввод операций без чисел
- Ввод только операций
- Ввод пробелов

### ФТ-07: Работа в браузере
**Источник:** БТ-05, UC-05
**Описание:** Система работает как клиентское веб-приложение
**Входные данные:** 
- Открытие страницы в браузере
**Ожидаемый результат:** 
- Калькулятор загружается и работает без серверной части
- Все функции доступны после загрузки
- Работает в автономном режиме после первой загрузки
**Граничные случаи:**
- Открытие в старых браузерах
- Отсутствие интернет-соединения после загрузки
- Обновление страницы (F5)
