# QuickCalc

Веб-калькулятор с базовыми арифметическими операциями и историей вычислений.

## Требования

- Браузер с поддержкой JavaScript и CSS
- Node.js и npm для запуска тестов

## Запуск

### Вариант 1: Открыть локально
1. Откройте файл `src/index.html` в браузере двойным кликом.

### Вариант 2: Запуск через HTTP-сервер
1. Перейдите в корневую директорию проекта.
2. Запустите HTTP-сервер:
   ```sh
   python -m http.server 8000
   ```
3. Откройте в браузере:
   ```
   http://localhost:8000/src/
   ```

## Структура проекта

- `src/app.js` — Основной файл приложения
- `src/calculator.js` — Логика калькулятора
- `src/index.html` — HTML-страница
- `src/keyboard.js` — Обработка ввода с клавиатуры
- `src/styles.css` — Стили приложения

## Тесты

1. Установите зависимости:
   ```sh
   npm install
   ```
2. Запустите тесты:
   ```sh
   npm test
   ```
