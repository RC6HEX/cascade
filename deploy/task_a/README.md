# QuickCalc: Веб-калькулятор

QuickCalc — это простой клиентский веб-калькулятор, позволяющий выполнять базовые арифметические операции (сложение, вычитание, умножение, деление). Приложение поддерживает историю последних 10 вычислений, ввод с клавиатуры и имеет тёмную тему по умолчанию.

## Требования

Для запуска приложения требуется современный веб-браузер (например, Chrome, Firefox, Edge, Safari).

## Запуск

Для запуска приложения выполните одно из следующих действий:

1.  **Открыть файл напрямую:**
    Откройте файл `src/index.html` в вашем веб-браузере.

2.  **Использовать локальный HTTP-сервер:**
    Из корневой директории проекта выполните команду:
    ```bash
    python -m http.server 8000
    ```
    Затем откройте в браузере адрес: `http://localhost:8000/src/`

## Структура проекта

```
.
├── src/
│   ├── app.js            # Основная логика приложения, инициализация UI и обработка событий.
│   ├── calculator.js     # Модуль, содержащий логику арифметических вычислений.
│   ├── history.js        # Модуль для управления историей вычислений.
│   ├── index.html        # Главный HTML-файл приложения.
│   └── styles.css        # Файл стилей для оформления пользовательского интерфейса.
├── tests/
│   └── app.test.js       # Тесты для основной логики приложения.
└── package.json          # Конфигурация проекта и зависимости для тестирования.
```

## Тесты

Для запуска тестов выполните следующие команды в корневой директории проекта:

1.  Установите зависимости (один раз):
    ```bash
    npm install
    ```
2.  Запустите тесты:
    ```bash
    npm test
    ```
