# CurrencyX: Конвертер валют

CurrencyX — это клиентское веб-приложение для конвертации валют в реальном времени. Оно позволяет пользователям выбирать исходную и целевую валюты, вводить сумму и мгновенно получать результат конвертации, используя актуальные курсы из внешнего API. Приложение поддерживает кеширование данных для работы в условиях отсутствия сети или недоступности API.

## Требования

Приложение CurrencyX реализует следующие функциональные и нефункциональные требования:

*   **Выбор валют:** Пользователь может выбрать исходную и целевую валюту из списка поддерживаемых.
*   **Актуальные курсы:** Отображение актуального курса обмена, полученного из `https://open.er-api.com/v6/latest/USD`.
*   **Мгновенная конвертация:** Результат конвертации обновляется в реальном времени при вводе суммы.
*   **Смена валют:** Возможность поменять местами исходную и целевую валюты одной кнопкой.
*   **Поддерживаемые валюты:** USD, EUR, RUB, GBP, JPY, CNY, TRY, KZT, BYN, GEL, CHF, CAD.
*   **Кеширование:** При недоступности API или отсутствии сети используются последние закешированные курсы из `localStorage` с пометкой о дате. Срок жизни кеша — 1 час.
*   **Веб-интерфейс:** Доступ через любой современный веб-браузер.
*   **Дизайн:** Тёмная тема с акцентным зелёным цветом (#10b981).
*   **Локализация:** Все надписи на русском языке.
*   **Архитектура:** Полностью клиентское приложение, не требующее серверного кода.

## Запуск

Для запуска приложения выполните одно из следующих действий:

1.  **Открыть в браузере:**
    Откройте файл `src/index.html` напрямую в вашем веб-браузере.

2.  **Использовать локальный веб-сервер (рекомендуется):**
    Убедитесь, что у вас установлен Python.
    Из корневой директории проекта выполните команду:
    ```bash
    python -m http.server 8000
    ```
    Затем откройте в браузере адрес: `http://localhost:8000/src/`
