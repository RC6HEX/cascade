# Юз-кейсы

### UC-01: Открытие страницы конвертера и загрузка курсов
**Источник:** БТ-02, БТ-06, БТ-07
**Актор:** Пользователь, Система
**Предусловие:** Пользователь имеет доступ к веб-браузеру и интернет-соединению.
**Основной поток:**
1. Пользователь открывает страницу конвертера в веб-браузере.
2. Система загружает актуальные курсы обмена из внешнего API.
3. Система кеширует полученные курсы.
4. Система отображает интерфейс конвертера, включая актуальный курс обмена (по умолчанию или для выбранных валют).
**Альтернативный поток — API недоступен:**
2a. Если внешний API не отвечает или недоступен, система использует последний закешированный курс.
2b. Система отображает рядом с курсом пометку: «Курс от [дата]. Актуальные данные временно недоступны».
**Постусловие:** Страница конвертера загружена, отображены курсы валют (актуальные или закешированные), пользователь готов к взаимодействию.

### UC-02: Выбор валют и ввод суммы для конвертации
**Источник:** БТ-01, БТ-03, БТ-05
**Актор:** Пользователь, Система
**Предусловие:** Страница конвертера загружена, курсы валют отображаются.
**Основной поток:**
1. Пользователь выбирает исходную валюту из выпадающего списка, содержащего минимум 10 популярных валют.
2. Пользователь выбирает целевую валюту из выпадающего списка, содержащего минимум 10 популярных валют.
3. Пользователь вводит сумму для конвертации в поле ввода.
4. Система в реальном времени (мгновенно) отображает сконвертированную сумму.
5. Система отображает текущий курс обмена между выбранными валютами.
**Постусловие:** Пользователь видит результат конвертации введенной суммы и актуальный курс обмена.

### UC-03: Обмен исходной и целевой валют местами
**Источник:** БТ-04
**Актор:** Пользователь, Система
**Предусловие:** Пользователь выбрал исходную и целевую валюты, и, возможно, ввел сумму для конвертации.
**Основной поток:**
1. Пользователь нажимает кнопку «⇄» (поменять местами).
2. Система меняет местами выбранные исходную и целевую валюты.
3. Если была введена сумма, система пересчитывает и мгновенно отображает новый результат конвертации с учетом поменянных валют.
4. Система обновляет отображаемый курс обмена в соответствии с новым направлением конвертации.
**Постусловие:** Исходная и целевая валюты поменяны местами, результат конвертации и курс обновлены.
