# TaskBoard

TaskBoard — это простое веб-приложение для отслеживания задач, реализованное в виде канбан-доски. Оно позволяет создавать задачи с названием и описанием, перемещать их между статусами («К выполнению», «В работе», «Выполнено») с помощью Drag-and-drop, удалять и фильтровать по статусу. Данные задач сохраняются в локальном хранилище браузера, обеспечивая их доступность между сессиями.

## Требования

Для работы приложения необходим современный веб-браузер.

## Запуск

Для запуска приложения выполните одно из следующих действий:

1.  Откройте файл `src/index.html` непосредственно в вашем веб-браузере.
2.  Запустите локальный HTTP-сервер из корневой директории проекта:
    ```bash
    python -m http.server 8000
    ```
    Затем откройте в браузере адрес: `http://localhost:8000/src/`

## Структура проекта

```
.
├── src/
│   ├── app.js            # Основная логика приложения, инициализация
│   ├── dragDrop.js       # Логика для функциональности Drag-and-Drop
│   ├── index.html        # Главный HTML-файл приложения
│   ├── storage.js        # Модуль для работы с локальным хранилищем
│   ├── styles.css        # Стилизация пользовательского интерфейса
│   ├── task.js           # Модель данных для задачи
│   └── ui.js             # Модуль для управления пользовательским интерфейсом
├── tests/
│   ├── storage.test.js   # Тесты для модуля storage.js
│   ├── task.test.js      # Тесты для модуля task.js
│   └── ui.test.js        # Тесты для модуля ui.js
└── package.json          # Конфигурация проекта и зависимости для тестов
```

## Тесты

Для запуска тестов:

1.  Установите зависимости (один раз):
    ```bash
    npm install
    ```
2.  Запустите тесты:
    ```bash
    npm test
    ```
