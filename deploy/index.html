Cascade

Автономный генератор приложений по бизнес-требованиям. Эта страница — три приложения, сгенерированные генератором по входным БТ + БП. Запусти их прямо в браузере или зайди в исходники / документацию.

Запустить генератор у себя: git clone репо ниже → bash install.sh → впиши OpenRouter ключ → python -m ui.
