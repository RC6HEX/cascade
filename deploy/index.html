Альфа Будущее Автономная команда разработки ПО

Генератор приложений по бизнес-требованиям

Каждая из этих демок сгенерирована автоматически из БТ + БП + Features. Генератор сам построил юз-кейсы, НФТ, ФТ, исходный код и тесты. Никакого ручного кода в output/.

A

QuickCalc

Веб-калькулятор. Базовые операции, обработка деления на ноль, история последних 10 вычислений, ввод с клавиатуры.
Запустить ФТ
B

TaskBoard

Канбан таск-трекер. Создание задач, drag-and-drop между статусами, фильтры, сохранение в localStorage.
Запустить ФТ
C

CurrencyX

Конвертер валют с курсами из открытого API. 12 валют, кеш в localStorage, fallback при недоступности API.
Запустить ФТ
Все три демки сгенерированы одним и тем же генератором (Python + Gemini API). Команда: python -m generator --task task_X
